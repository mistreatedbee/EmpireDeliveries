# Empire Deliveries — Mobile (Expo)

This folder contains a scaffold for an Expo app using Expo Router and NativeWind.

Quick start:

1. cd mobile
2. npm install
3. npm run start

Notes:
- Screens are placeholders — you should convert the React web components in `src/components` into React Native equivalents under `mobile/app/`.
- Backend base URL is configured in `mobile/app/config.ts` and points to `https://insforge.dev`.
- NativeWind (Tailwind for RN) is configured via `mobile/tailwind.config.js` and `babel.config.js`.
